Bamazon

BAMAZON is an Amazon-like storefront. Customers have the ability to browse and purchase through several items in different departments ranging from Home & Garden to Outdoor & Sports. Managers have the ability to "View Products", "View Low Inventory", "Add to Inventory" and "Add New Product". Built with Javascript, Node.js and MySQL.

Liri

LIRI is a command line application built with Node.js that you can run in your terminal.

  • Search Spotify for any song.
  • Find upcoming concerts by using "Bands in Town".
  • Search OMDB (the Open Movie Database) for any movie.
  • Surprise you with one of the above.
